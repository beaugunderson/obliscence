Search past Claude Code conversations using obliscence.

Commands:
- Search: `obliscence search "$QUERY" --json`
- Filter by project: `obliscence search "$QUERY" --project NAME --json`
- Filter by role: `obliscence search "$QUERY" --role user --json` or `--role assistant`
- Filter by date: `obliscence search "$QUERY" --after 2026-01-01 --before 2026-04-01 --json`
- List sessions: `obliscence sessions --json --limit 20`
- List sessions for a project: `obliscence sessions --project NAME --json`
- Show a session: `obliscence show SESSION_ID_OR_SLUG`
- List projects: `obliscence projects --json`
- Stats: `obliscence stats --json`
